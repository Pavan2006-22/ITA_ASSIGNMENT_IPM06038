## Project Title
Identifying Fake Product Reviews on E-Commerce Platforms

This project focuses on detecting fake product reviews in an Amazon reviews dataset using machine learning techniques such as **K-Means Clustering** and **Logistic Regression**. The analysis includes feature engineering, sentiment analysis, clustering to detect suspicious review patterns, and classification to predict whether a review is fake or genuine.
